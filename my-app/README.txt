React Front-end

----------------------------------------------------------------------------------------------
Installation ---------------------------------------------------------------------------------
----------------------------------------------------------------------------------------------
From the root of my-app run the following command to install all node modules

npm install

*For Front-end to speak to back-end I needed to make sure to have Allow-Control-Allow-Origin chrome extension installed and enabled. 
https://chrome.google.com/webstore/detail/allow-cors-access-control/lhobafahddgcelffkeicbaginigeejlf?hl=en

----------------------------------------------------------------------------------------------
Start React Front-end ------------------------------------------------------------------------
----------------------------------------------------------------------------------------------
From the root of my-app run the following command

npm start
